# AI Job Intelligence Platform

An AI-powered platform that helps job seekers analyze job trends, match resumes with job descriptions, predict ATS scores, identify skill gaps, and receive AI-driven resume improvement suggestions.

## Tech Stack
- Python
- SQL
- Pandas
- Streamlit
- Power BI
- Scikit-learn
- Git & GitHub
## Project Roadmap

### Module 1: Data Collection & Analysis
- Load job dataset
- Clean data
- Perform exploratory data analysis
- Visualize job trends

### Module 2: Database
- Store jobs in SQLite
- Write SQL queries

### Module 3: Job Scraper
- Scrape jobs from websites
- Update database automatically

### Module 4: Resume Intelligence
- Resume parsing
- Resume vs Job matching

### Module 5: AI Features
- ATS score prediction
- Skill gap analysis
- AI resume suggestions

### Module 6: Dashboard & Deployment
- Streamlit app
- Power BI dashboard
- Deploy online
